# Chicago Star

Six-pointed star designs inspired by the Chicago city flag — coasters, mesh art pieces, and decorative prints.

## Overview

Multiple design iterations of the iconic Chicago star, evolving from flat outlines to 3D mesh/grid-style coasters. Modeled in Blender with exported STL/3MF files ready for slicing.

## Files

### Source Models (Blender)

- `star.blend` / `star2.blend` / `star3.blend` — Star iterations (solid forms)
- `star3a.blend` / `star3a-simple-grid.blend` — Grid/mesh pattern variants
- `star6.blend` / `star7.blend` — Later design explorations
- `starmesh4.blend` / `starmesh5.blend` — Mesh-style star coasters
- `newstar2.blend` — Revised star design

### Print-Ready Files

- `star3-cut-coaster*.3mf` — Star coaster variants (multiple iterations) with slicer settings
- `starmesh4-mesh-with-rough-coaster.3mf` — Mesh-style star coaster
- `newstar2.3mf` — Revised star print project

### Exports (STL)

- `Star.stl` / `Cylinder.stl` / `Disc.stl` — Component meshes
- `star3.stl` / `star3-orcaexport.stl` — Star v3 exports
- `starmesh3.stl` / `starmesh4.stl` / `starmesh5.stl` — Mesh star exports
- `newstar2.stl` — Revised star export

### Reference

- `outline.svg` — Chicago star vector outline
- `ChicagoStarMesh.pptx` — Design notes / presentation
- `FASTPANEL.jpeg` — Reference image

## Tools

- Blender
