# Pentaho Logo

3D-printable Pentaho logo — extruded from the official logo SVG into a solid display piece.

## Overview

The Pentaho logo converted from 2D vector art into 3D-printable forms using Blender. Includes both flat (relief-style) and full 3D variants.

## Files

### Source

- `pentaho-2.svg` — Vector logo artwork
- `pentaho-logo-png_seeklogo-300213.png` — Reference logo image

### Source Models (Blender)

- `pentahologo.blend` — Logo import/setup
- `pentaho1.blend` — Main logo model
- `pentaho2.blend` — Alternate version

### Print-Ready

- `pentaho-logo-flat.3mf` — Flat/relief logo slicer project
- `pentaho-logo.3mf` — Full 3D logo slicer project
- `pentaho-full.3mf` — Complete assembly slicer project

### Exports (STL)

- `pentaho-logo-flat.stl` — Flat relief export
- `pentaho-logo.stl` — 3D logo export
- `pentaho1.stl` — Logo v1 export

## Tools

- Blender (SVG import → mesh extrusion)
