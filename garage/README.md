# Garage Header

Custom garage door header trim pieces designed to fill gaps or finish exposed framing above a garage door.

## Overview

Multiple iterations of a header trim piece, modeled in Blender. Includes base and cap variants for a multi-part assembly.

## Files

### Source Models (Blender)

- `garage-header-1.blend` — Initial header design
- `garage-header-1b.blend` — Revised header (variant B)
- `garage-header-2.blend` — Second header design
- `garage-header-3.blend` — Third header design

### Print-Ready

- `garage-header-1b.3mf` — Slicer project for header variant 1B

### Exports (STL)

- `garage-header-1.stl` — Header v1
- `garage-header-1b.stl` — Header v1B (full piece)
- `garage-header-1b-base.stl` — Header v1B base component
- `garage-header-1c.stl` — Header v1C variant
- `garage-header-2.stl` — Header v2

## Tools

- Blender
