# Gridfinity Rationell

A Gridfinity-compatible adapter for IKEA Rationell drawer organizer systems.

## Overview

Bridges the [Gridfinity](https://gridfinity.xyz/) modular storage ecosystem with IKEA's Rationell drawer insert system, allowing Gridfinity bins to sit securely inside Rationell trays.

## Files

- `GridfinityRationell.3mf` — Print-ready 3MF project with slicer settings

## Tools

- OrcaSlicer / Bambu Studio
